Behind The Scenes - Quick Notes

My requirements for this web site are:

  • Web site will be client-based only.
  • Emphasise content, with minimal eye candy.
  • Design, Layout and coding by author to maximize learning opportunity.
  • Defer content suitable for blog or Twiki for later.

This website is version controlled using Git.

An example of a browser dependent page is the Software Process Automation page. Moving the mouse over the rectangular subgraphs will change the cursor to "+" crosshairs for Internet Explorer 7 and Firefox but not for Opera and Mozilla. Click on the "reduced functionality" for more details.
