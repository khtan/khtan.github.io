This document will step you through the process of getting started.  Steps 1, 2, and 3 can be done before the first official day of the course.

 

 

 

 

Step 1: Login to your my.pcc.edu account

Approximately 5 days after you officially register for the course, go to   my.pcc.edu  and login to your myPCC account.  (If this is your first time at the site, instructions are given on how to login.)

 

When you are logged in, check your My E-mail Inbox for a message that contains your Microsoft Academic Alliance login, password and URL for the site.  You will use this information in the next step.

 

 

 

Step 2: Get the software

You are going to need this software for the course: Microsoft Visual Studio 2003.  The good news is that you can either download it for free or buy a CD at a nominal charge from the Microsoft Academic Alliance website. 

 

Login to the website using the info from Step 1.

 

Choose the Visual Studio .NET 2003 Professional software. The download is free. The CD costs a few dollars. But students recommend the CD because the download is so large.  Please read the system requirements for this software to make sure your system will support it.  Install the software.

 

 

 

Step 3: Buy the Textbook



For the C# language:   "Visual C# .NET Step by Step"; Jagger and Sharp; Microsoft Press.

 

This book is optional:
For ADO .NET Database:  "Beginning C# Databases"; WROX press.

 

Step 4: Login to the course website

 

This step cannot be done until the first official day of the course. On the first official day of the course you can login to the course website.

 

The URL is:   http:\\webct.pcc.edu

 

To find out what your userid and Password are, click on the link "What is my WebCT ID & Password?"  

 

Once you are logged on to the website there will be information about the course. 

 

The materials on the website consists of:

  • lecture notes
  • assignments
  • exams
  • code samples

 

 

It is important to mention that some of you registered for the online section and others for the on campus section.  No matter which section you registered for you will need to use the website for the course materials. That is, everyone needs to use the course website, no matter if you are an on campus or online student.
