Course Website Information

  C# Programming (CIS234N)

 

 

 

All of the materials for the course - assignments, lecture notes, code samples and the final exam - are on the course website.  All students must use the site to take the exam, turn in the assignments and access the course materials.

 

 

Homepage

 

The Homepage will have icons for  Syllabus, Course Materials, Assignment Box, Private Mail, Class Discussion, My Grades and other links.

 

 

 

Class Discussion

 

Use the Class Discussion board to post questions to all the students about the course, labs, lecture topics, etc.

 

Note: During the first week of class you must compose an introductory message using the Class Discussion board. Post it to the Student Introductions topic. Just say hello so I know you showed up for the course. Optionally, you can tell a little about yourself.

 

 

 

Course Material

 

The course material is provided as a zipped (compressed) folder named CIS234N.  The folder is on the course website. You will find its link on the Homepage. Download the folder and uncompress it.  Inside the folder open the CourseMaterialIndex web page.

 

 

 

 

Private mail

 

The Private Mail facility is used to email me or other students in the class.  This is a closed email system and has no connection to other email, so you can only email students in the course and the instructor.  Use the Browse button to select the recipient.

 

 

 

 

Assignments Box

 

 

The Assignments Box will have all the assignments for the course. Click on the assignment to get access to it. 

 

Note: You must submit your projects to the Assignments Box . Each assignment will each have an Availability deadline; submit the project before the deadline expires. The assignment box will not accept submissions after the Availability ends.

 

After your project is graded you can see your grade and my comments.  The grade will also appear in your My Grades feature.

 

 

 

 

 

Exams

 

The Exams page will have the Final Exam for the course. The exam will be available for the period listed. You must take the exam and complete it within that time period. You will see the dates next to the name of the exam.

 

 

 

My Grades

 

This link will have your grades posted. Your grades are private; only you will be able to see them.
